JavaScript blocos de código

instruções JavaScript podem ser agrupadas em blocos de código, dentro de chaves {...}.

O propósito dos blocos de código é definir instruções a serem executadas em conjunto.

JavaScript Palavras-chave

Instruções JS muitas vezes começam com uma palavra-chave para identificar a ação JavaScript para ser executado.

  • break Finaliza um interruptor ou um loop
  • continue Salta para fora de um ciclo e começa no topo
  • debuggerInterrompe a execução de JavaScript, e chamadas (se disponível) a função de depuração
  • do ... while Executa um bloco de instruções, e repete o bloco, enquanto uma condição for verdadeira
  • for Marca um bloco de instruções a serem executadas, desde que uma condição é verdadeira
  • function Declara uma função
  • if ... else Marca um bloco de instruções a serem executadas, dependendo de uma condição
  • return Sai uma função
  • switch Marca um bloco de instruções a serem executadas, dependendo de diferentes casos
  • try ... catch Implementa o tratamento de erros para um bloco de instruçõe
  • var Declara uma variável
